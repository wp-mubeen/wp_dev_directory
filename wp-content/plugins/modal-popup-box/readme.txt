== Description ==

= Modal Popup Box - Popup Builder =

Modal Popup Box is enable the functionality to add pop content on your WordPress website. You can add and show any type of content into modal pop up box. You can customize the popup box title, height and width, color, loading effect using plugin setting panel. You can enable popup functionality on button click or page load.

You can embed and display anything in popup box like image, YouTube videos, Vimeo videos, image gallery, album gallery, photo album, slider slideshows, various plugin shortcodes, tables, pricing tables, social media feeds, Facebook like box, Twitter tweets, Flickr feeds, Contact Form 7, buttons, HTML contents, greeting messages and images, coupon codes, discount codes, free and promotional offers, social media icons, customer feedback, clients testimonials, general forms, registrations forms.

== Installation ==

Install Modal Pop Box builder either via the WordPress.org plugin directory or by uploading the files to your server.

After activating Modal Pop Box maker, go to plugin menu.

Click Add Modal Pop Box and write the content in the text box.

Publish the modal pop box builder and copy shortcode from the bottom of the add images box and embed shortcode on any Page/Post/Text Widget.

That's it. You're ready to go!
